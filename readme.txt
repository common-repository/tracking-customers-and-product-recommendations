Product recommendations to your customers

== Description ==
This plugin makes it easier for your customers to reach the products they care about.

<h4>Plugin information</h4>
<ul>
    <li>Tracking customers</li>
    <li>Product recommendations</li>
    <li>Widgets support</li>
</ul>

== Installation ==
1. Upload the entire plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
